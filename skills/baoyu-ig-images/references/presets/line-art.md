# Line-Art Style

Mostly white background, single ink color + 1-2 pastels, illustrated — educational and clear.

## Element Combination

```yaml
canvas:
  ratio: portrait-4-5
  grid: single | dual | triptych

image_effects:
  cutout: stylized
  stroke: white-solid
  filter: none | desaturated-matte

typography:
  decorated: none | highlight
  tags: pill | box-outline
  direction: horizontal

decorations:
  emphasis: arrow-minimal | numbering | circle-mark
  background: solid-white | grid-subtle
  elements: minimal-lines | geometric-shapes
  frames: thin-outline | rounded-rect
```

## Color Palette

| Role | Colors | Hex |
|------|--------|-----|
| Primary | Ink black or dark navy (single color) | #2C3E50 or #1A1A2E |
| Background | Clean white, very light gray | #FFFFFF, #FAFBFC |
| Accents | 1-2 soft pastels (light blue, soft coral) | #B8D4E3, #F5C6AA |

## Visual Elements

- Single-color line illustrations
- Clean hand-drawn diagrams and icons
- Consistent stroke weight throughout
- Minimal use of fills (mostly outlines)
- Educational diagrams and visual explanations
- Simple, clear iconography

## Typography

- Clean hand-lettering or monoline style
- Single consistent stroke weight
- Minimal decoration on text
- Clear hierarchy through size only (not weight)
- Labels and annotations feel hand-written
