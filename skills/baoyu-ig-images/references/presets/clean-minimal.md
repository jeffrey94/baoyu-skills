# Clean-Minimal Style

White space, structured, professional — the polished Instagram aesthetic.

## Element Combination

```yaml
canvas:
  ratio: portrait-4-5
  grid: single | dual

image_effects:
  cutout: clean
  stroke: none | white-solid
  filter: none | desaturated-matte

typography:
  decorated: none | mixed-weight
  tags: pill | box-outline
  direction: horizontal

decorations:
  emphasis: underline | dot-bullet
  background: solid-white | neutral-warm
  elements: geometric-shapes | minimal-lines
  frames: rounded-rect | thin-outline
```

## Color Palette

| Role | Colors | Hex |
|------|--------|-----|
| Primary | Black, charcoal | #111111, #333333 |
| Background | White, off-white, light gray | #FFFFFF, #FAFAFA, #F5F5F5 |
| Accents | 1-2 accent colors (muted blue, sage, terracotta) | #6B8FAD, #8FA68B, #C4846C |

## Visual Elements

- Clean geometric shapes, thin lines
- Maximum whitespace, structured grid
- Subtle shadows for depth
- Minimal decoration — content speaks

## Typography

- Clean sans-serif (Inter, Helvetica Neue feel)
- All-caps tracked headlines
- Consistent hierarchy with generous spacing
- One bold element per slide
