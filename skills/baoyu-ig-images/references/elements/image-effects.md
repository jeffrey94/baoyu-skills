# Image Processing Layer

Visual effects applied to image elements in Instagram carousel infographics.

## AI Cutout

Subject extraction styles for product/figure isolation.

| Name | Description | Use Case |
|------|-------------|----------|
| clean | Sharp edges, precise boundaries | Product photography, tech items |
| soft | Soft transition, feathered edges | Portrait cutout, organic subjects |
| stylized | Illustrated/artistic edge treatment | Creative compositions |

## Stroke Effects

Border treatments for cutout elements.

| Name | Description | Use Case |
|------|-------------|----------|
| white-solid | White solid line border | Clean sticker feel, high contrast |
| colored-solid | Colored solid line border | Brand colors, playful vibe |
| dashed | Dashed/dotted border | Handmade aesthetic, casual |
| double | Double-layer stroke | Emphasis, premium feel |
| glow | Soft outer glow | Dreamy, neon aesthetic |
| shadow | Drop shadow effect | Depth, floating element |

**Stroke Width Guidelines**:
- Thin: 2-4px - Subtle, elegant
- Medium: 5-8px - Standard visibility
- Thick: 10-15px - Bold emphasis

## Texture Overlays

Additional texture effects.

| Name | Description | Use Case |
|------|-------------|----------|
| paper | Paper or craft texture | Organic, handmade feel |
| noise | Fine grain noise | Analog aesthetic, depth |
| halftone | Dot pattern | Retro print style |
| linen | Subtle linen/fabric texture | Premium, tactile |

## Blending Modes

For layered compositions.

| Mode | Effect | Use Case |
|------|--------|----------|
| multiply | Darken, merge | Shadow effects |
| screen | Lighten, glow | Light effects |
| overlay | Contrast boost | Vibrant compositions |
| soft-light | Subtle blending | Natural layering |

## Effect Combinations

Common effect stacks for different styles:

### Clean-Minimal Style
- Filter: none or desaturated-matte
- Stroke: none or white-solid (thin)
- Texture: none

### Bold-Graphic Style
- Filter: high-contrast
- Stroke: colored-solid (thick) or glow
- Texture: none or noise (subtle)

### Soft-Muted Style
- Filter: soft-glow or desaturated-matte
- Stroke: white-solid (medium)
- Texture: paper (subtle)

### Editorial Style
- Filter: high-contrast or cinematic-teal-orange
- Stroke: none or white-solid (thin)
- Texture: none or noise (subtle)

### Retro-Nostalgia Style
- Filter: vintage-fade or film-grain-warm
- Stroke: dashed or double
- Texture: noise, halftone

### Neon-Gradient Style
- Filter: duotone or high-contrast
- Stroke: glow
- Texture: noise (subtle)

### Earthy-Organic Style
- Filter: warm-tone or film-grain-warm
- Stroke: none or dashed
- Texture: paper, linen

### Dark-Cinematic Style
- Filter: cinematic-teal-orange or high-contrast
- Stroke: glow or shadow
- Texture: noise (subtle)

### Playful-Pop Style
- Filter: none (already vibrant)
- Stroke: colored-solid (thick) or white-solid
- Texture: none

### Line-Art Style
- Filter: none or desaturated-matte
- Stroke: white-solid (thin) or none
- Texture: paper (subtle)
