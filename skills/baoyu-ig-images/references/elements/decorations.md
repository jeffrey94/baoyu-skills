# Decorative Assets

Visual embellishments and decorative elements for Instagram carousel infographics.

## Emphasis Marks

Elements to draw attention to specific content.

| Name | Description | Use Case |
|------|-------------|----------|
| arrow-minimal | Clean minimal arrow | Pointing to features, direction |
| circle-mark | Circle highlight annotation | Highlighting details |
| underline | Straight or accent underline | Text emphasis |
| checkmark | Checkmark/tick symbol | Completed items, pros |
| cross-mark | X mark symbol | Cons, things to avoid |

| bracket | Bracket highlighting | Grouping, emphasis |

## Style-Specific Decorations

### Clean-Minimal Style
- Geometric shapes, thin lines
- Maximum whitespace
- Subtle shadows, no clutter

### Bold-Graphic Style
- Strong geometric forms, neon accents
- High contrast elements
- Dynamic angles, bold shapes

### Soft-Muted Style
- Organic blobs, soft gradients
- Pastel overlays
- Gentle, flowing shapes

### Editorial Style
- Editorial frames, thin rules
- Elegant dividers
- Magazine-quality layout elements

### Retro-Nostalgia Style
- Film grain, light leaks
- Vintage badges, retro shapes
- Aged texture overlays

### Neon-Gradient Style
- Gradient overlays, glow effects
- Glass morphism, mesh gradients
- Digital, futuristic elements

### Earthy-Organic Style
- Botanical elements, brush strokes
- Paper textures, organic shapes
- Natural, handcrafted feel

### Dark-Cinematic Style
- Vignette, dramatic lighting
- Jewel-tone accents on dark bg
- Elegant serif + thin rules

### Playful-Pop Style
- Bold shapes, bright colors
- Confetti, stars, energetic elements
- Rounded, bubbly containers

### Line-Art Style
- Single-color illustrations
- Clean hand-drawn elements
- Minimal, educational clarity
